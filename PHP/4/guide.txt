STEP 1: Use same DB

USE student_db;

STEP 2: Create Table

CREATE TABLE employees (
    emp_id INT AUTO_INCREMENT PRIMARY KEY,
    emp_name VARCHAR(100),
    department VARCHAR(100),
    salary INT
);

STEP 3: Run Project

1. Put folder in htdocs
2. Start Apache + MySQL
3. Open:
   http://localhost/employee-ajax/index.php

IMPORTANT:
- AJAX using Fetch API (no page reload)
- Update only salary (as per question)
- Same database used
